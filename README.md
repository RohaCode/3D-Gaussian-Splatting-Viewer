# 🌌 Panorama: 3D Gaussian Splatting Viewer

Интерактивный, высокопроизводительный веб-просмотрщик файлов **3D Gaussian Splatting** (`.splat` и `.spz`), оптимизированный для мгновенной загрузки, нулевой серверной нагрузки и работы на **GitHub Pages**.

## ✨ Ключевые особенности

*   **⚡ Умный On-Demand Рендеринг (0% нагрузки на GPU)**: Встроенный детектор движения отслеживает изменения камеры и параметров. Когда вы не двигаете камеру и не настраиваете сцену, рендеринг WebGL автоматически засыпает, мгновенно снижая нагрузку на видеокарту до нуля.
*   **🎯 Полная конфиденциальность (Client-side)**: Все импортируемые пользователем `.splat` и `.spz` файлы обрабатываются локально прямо в браузере с помощью URL-объектов. Никакие данные не отправляются на удаленный сервер.
*   **💎 Премиальный стеклянный интерфейс (Glassmorphic UI)**: Элегантная темная панель управления с мягким свечением, размытием заднего фона и современными слайдерами.
*   **🌍 Автоматическая загрузка по ссылке**: Поддержка параметров запроса `?file=path.splat`. Делитесь ссылкой, и сцена загрузится автоматически!
*   **📐 Профессиональные инструменты выравнивания**:
    *   Регулировка высоты (ось Y)
    *   Полный контроль вращения во всех 3 плоскостях (Pitch, Yaw, Roll)
    *   Быстрый экспорт параметров в JS-код в один клик.
*   **🔒 Ограничения камеры (Camera Boundaries)**: Настроены жесткие рамки вращения и зума (OrbitControls), которые не позволяют камере "улетать" за пределы стен или переворачиваться.

## 🛠 Технологический стек

1.  **Ядро 3D**: `gsplat.js` (загрузка через надежный CDN ES-модуль, без лишних npm-зависимостей).
2.  **Стили**: Ванильный CSS3 с адаптивной версткой, кастомными шрифтами (Inter, Outfit) от Google Fonts и анимациями.
3.  **Логика**: Модульный чистый ES6 JavaScript (`app.js`).
4.  **Сервер**: Легковесный Node.js сервер (`server.js`) для локальной отладки.

## 🚀 Локальный запуск (Node.js)

Для запуска локального сервера статических файлов с правильными MIME-типами (включая `.splat` и `.spz`):

1. Убедитесь, что у вас установлен **Node.js** (версии 14 и выше).
2. Перейдите в папку `3D-Gaussian-Splatting-Viewer`:
   ```bash
   cd 3D-Gaussian-Splatting-Viewer
   ```
3. Установите зависимости (проект не требует внешних пакетов, используется стандартная библиотека `http`):
   ```bash
   npm install
   ```
4. Запустите локальный сервер:
   ```bash
   npm start
   ```
5. Откройте в браузере адрес:
   [http://localhost:8000](http://localhost:8000)
